### Các bước chạy dự án

1. Tải code về\
   `git clone <link dự án>`
2. Cài Node.js
3. Cài các thư viện\
   `npm i`
4. Chạy dự án (tại localhost:5178, hoặc xem trên terminal)\
   `npm run dev`

### Quy tắc
